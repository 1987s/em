1. Identifiers:
   - Examples: variable, functionName, ClassName, enumCase, camelCase

2. Keywords:
   - Examples: if, else, for, while, class, struct, enum, func, let, var, return

3. Literals:
   - Numeric Literals:
     - Examples: 42, 3.14, 0b1010, 0o777, 0xFF, 1_000_000
   - String Literals:
     - Examples: "Hello, World!", "Swift is awesome"
   - Character Literals:
     - Examples: 'A', 'b', '\n', '\u{1F600}'
   - Boolean Literals:
     - Examples: true, false
   - Array Literals:
     - Examples: [1, 2, 3], ["a", "b", "c"]
   - Dictionary Literals:
     - Examples: ["key1": "value1", "key2": "value2"] and [1: "One", 2: "Two"]

4. Operators:
   - Arithmetic Operators:
     - Examples: +, -, *, /, %,
   - Assignment Operators:
     - Examples: =, +=, -=, *=, /=, %=
   - Comparison Operators:
     - Examples: ==, !=, <, >, <=, >=
   - Logical Operators:
     - Examples: &&, ||, !
   - Bitwise Operators:
     - Examples: &, |, ~, ^, <<, >>

5. Delimiters:
   - Examples: (, ), {, }, [, ], ,, ;

6. Comments:
   - Single-line Comment:
     - Example: // This is a comment
   - Multi-line Comment:
     - Example:
       swift
       /*
       This is a
       multi-line comment
       */


7. Control Characters:
   - Examples: \n, \t, \r, \\, \", \'

8. Whitespace:
   - Examples: Space ( ), Tab (\t), Newline (\n), Carriage Return (\r)

9. End-of-Line:
   - Example: ;

Please note that this is not an exhaustive list, and there may be additional token types depending on the context and language features used in the Swift code. This list covers some of the most common token types encountered in Swift programming.
